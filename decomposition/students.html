Students in a Room

It’s very likely that you already practice decomposition intuitively in your day to day without realising. Consider the following theatre set up. How many students can you fit seated in this room? Each box represents a desk with 2 chairs. How did you figure that out?

Using a computational thinking approach:

  • 1 Student == 1 Chair
  • 2 Chairs == 1 Box
  • 7 boxes = 1 row
  • 5 rows == 1 room.
5 * 7 * 2 = 70 students can sit in this room. Let’s expand this problem further.


Solution
Using your new decomposition skills, you would likely divide the room into 3 sections and then calculate the number of seats in section A + section B + section C. Follow the same workflow identified in Example 2, for each respective section. Then add A + B + C.
