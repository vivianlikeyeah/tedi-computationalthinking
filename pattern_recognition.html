Pattern Recognition

Moving onto Pattern Recognition
Let’s run through our Minecraft example again:

  • We obtain some wood.
    • We make a crafting table.
    • We make a tool.
    • We make some coal.
    • We farm some animals.
    • We cook our meat.
  • Then we need our light source.
    • We obtain some wood.
    • We make a crafting table.
    • We make a tool.
    • We obtain some coal.
    • We make some torches.
  • Then we build a roof/dig our hole
    • We obtain some wood.
    • We make a crafting table.
    • We make a tool.
    • We build a roof.

When looking at our Minecraft Example, something stands out a little. To obtain food, light and shelter, we need to make tools. In order to make tools, we need wood. So for all three very different tasks, there are several tasks that are in common.

Rather than go through all of process 1, then go through all of process 2, and all of process 3, wouldn’t it be faster if we just obtained all the wood at once?

How did we realise we can do this? We realised it through a process of pattern recognition. We noticed that all processes start with obtain wood – this is a pattern.

Recognising patterns is incredibly useful in computational thinking, as it allows you to develop efficient programs.

Back to the basics
All pattern recognition, as with many concepts in engineering, comes back to the basics. Some of these problems you’re very likely to have encountered throughout high-school (or even primary school). But fear not if you haven’t, we’ll go through the way to solve them regardless. It’s less important to get the right answer, and more important to understand how do we come to that answer.

  1. What is the next number in this sequence?
    {1,3,5,7,9}

    Answer:
    11. Each number is the last number + 2
  2. What is the next number in this sequence?
    {2,4,6,8}

    Answer:
    10. Each number is the last number + 2
  3. What is the next number in this sequence?
    {27,22,17,12}

    Answer:
    7. N = N – 5
  4. What is the next shape in this sequence?
    (Yellow, Black, Yellow, Black, Yellow, Black)
  5. What is the next shape in this sequence?
    (2 sides, 3 sides, 4, sides, 5 sides)
  6. What is the next animal in this sequence?
    (2 legs, 4 legs, 6 legs)
